📊 Chart Library
Ten chart types with anatomy annotations and ❌/✅ do/don't pairs. Each chart implements a specific task from the Visual Selection Decision Framework v3.
Chart Type 1 — Horizontal Bar Chart · Task: Comparison (Ranked Items)
Anatomy
Horizontal Bar — Monthly Visits (sorted descending)
① Sort descending by value — enables immediate rank reading
② Zero baseline mandatory — Lie Factor = 1.0
③ Range-frame axis: tick at min value, tick at max value only
④ Label inside bar if bw > 72px, else outside — no axis needed
⑤ Single hue: bar/main #4a4845 — no color legend required
Do / Don't
Common Bar Chart Errors
✅ Do — Sorted, zero-based, direct label
❌ Don't — Non-zero baseline (Lie Factor > 1)
Chart Type 2 — Slope / Line Chart · Task: Trend Over Time
Anatomy
User Growth 2023→2025 — Slope chart, log Y, shared scale
① Year guide lines: border/default 0.5px — visual structure without chartjunk
② Log Y scale when values span multiple orders of magnitude
③ End-value labels replace axis ticks — direct, no legend
④ No dots on data points — line alone is sufficient
⑤ Dashes distinguish overlapping series without color legend
Do / Don't
Spaghetti vs. Clarity
✅ Do — Label ends, no dots, max 5 series
❌ Don't — Spaghetti with legend (eye travel)
Chart Type 3 — Scatter Plot · Task: Relationship / Correlation
Anatomy
Revenue vs. Monthly Visits — range-frame axes, direct labels
① Outlier labels call out anomalies — Claude: high rev / low visits
② Range-frame: axis tick only at min and max data values
③ Regression line: border/default dashed — context, not decoration
④ Direct labels on dots — no legend required
Do / Don't
Scatter Errors
✅ Do — Range-frame, direct labels
❌ Don't — Bubble (area decodes as diameter²)
